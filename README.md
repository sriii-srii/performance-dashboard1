# Real-time Performance Dashboard

## Features
- Line, Bar, Scatter, Heatmap charts (canvas+SVG, no chart.js/D3)
- Real-time streaming and filtering
- Pan/zoom on line charts
- Data aggregation controls
- Performance monitoring (FPS, memory)
- Responsive design for desktop, tablet, mobile

## Setup
npm install
npm run dev

Visit: http://localhost:5173

## How to use
- Filter data by category and value
- Select time range for aggregation
- View live charts and table
- Monitor performance at top
